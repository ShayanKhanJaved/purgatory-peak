# Why Open Source?

Purgatory Peak is committed to the open-source philosophy. Here’s why we believe this game should be free and accessible to everyone:

1.  **Community-Driven Innovation**: Open-sourcing allows developers from around the world to contribute new features, fix bugs, and improve the game’s performance. Collaborative development leads to a better product for everyone.
2.  **Transparency and Security**: By making the source code available, players and developers can verify the code themselves. This ensures that the game remains secure and free from any hidden malicious intent.
3.  **Educational Asset**: For aspiring game developers, seeing a functional 2D survival game in Godot can be a valuable learning resource. We want to empower others to learn and create.
4.  **Free Access for All**: We believe that quality entertainment and creative tools should be available to everyone, regardless of their financial situation. Keeping this game free to use ensures that no one is barred from experiencing it.
5.  **Preservation**: Proprietary games can vanish if the developer stops supporting them. Open-source games live on through the community, ensuring the project remains playable and adaptable for years to come.
