# Purgatory Peak

Purgatory Peak is a 2D survival game where players must survive high-stakes encounters. The goal is to outrun hostile sprites that hunt you down while collecting as many coins as possible.

## Gameplay
- **Objective**: Surive the hostile sprites and gather coins.
- **Controls**: WASD/Arrow keys to move.

## Installation and Execution (Ubuntu)

To run this Godot game on Ubuntu, follow these steps:

1.  **Install dependencies**:
    ```bash
    sudo apt update
    sudo apt install -y g++ mesa-utils x11-utils libasound2-dev libudev-dev libxi-dev libxcursor-dev libxinerama-dev libxrandr-dev
    ```

2.  **Download Godot Engine**:
    Download the Godot Engine 4 (Linux 64-bit) from the [official website](https://godotengine.org/download/linux/).

3.  **Run the game**:
    Navigate to the project directory and run:
    ```bash
    /path/to/Godot_v4.x-stable_linux.x86_64 project.godot
    ```
